## BMI Data Analysis & Health Distribution Report
An end-to-end data analytics project focused on gathering physical health metrics and visualizing body mass index distributions across a sample population.

##  Business Logic
The goal was to categorize individuals into health risk groups based on World Health Organization (WHO) standards.

##  Tool Stack
- **Collection:** Google Forms
- **Processing:** Microsoft Excel (Advanced Formulas & Data Cleaning)
- **Visualization:** Microsoft Power BI (DAX, Interactive Slicers)

##  Key Metrics Visualized
- **BMI Distribution:** Count of individuals in each WHO category.
- **Gender-wise Analysis:** Average BMI comparisons between genders.
- **Correlation:** Relationship between age groups and health categories.
